# Little St. James Island - Map Layout

## Overview

The island (nicknamed "Pedophile Island" or "Orgy Island") is located in the U.S. Virgin Islands, southeast of St. Thomas. It's approximately 70 acres with a crescent/boomerang shape.

## 3D Model Coordinates

The in-game world uses a coordinate system where:
- **Center (0, 0)**: Central grounds area
- **X-axis**: West (-) to East (+)
- **Z-axis**: North (-) to South (+)
- **Y-axis**: Elevation

Total playable area: ~500 x 300 units

## Key Locations

### The Temple (-180, -80)
**The most infamous structure on the island.**

- Blue and white striped exterior
- Gold dome on top
- Located on elevated western ridge
- Dimensions: 20x25x20 units
- Flanked by owl/bird statues
- Underground levels (not accessible in current build)

Historical notes:
- Built between 2009-2013
- Originally had a golden dome (later painted blue)
- Purpose remains unknown/disputed
- Entrance faces south

### Main Residence (50, 0)
**The primary living quarters.**

- Large multi-story mansion
- Central location on island
- Dimensions: 60x15x40 units
- Multiple guest rooms
- Connected to all major paths

### Guest Villas
**Villa A (120, 60)** - Larger guest house, eastern area
**Villa B (150, 20)** - Secondary guest quarters

### Staff Quarters (-50, 80)
- Northern area
- Lower-profile building
- Evidence suggests monitoring equipment was housed here

### Sundial Plaza (0, -60)
- Decorative sundial/compass structure
- Central meeting point
- Connected to temple path

### Helipad (180, -40)
- Eastern edge of island
- Primary arrival point for VIPs
- Security station nearby

### Security Station (200, 0)
- Monitors island approaches
- Communication hub
- Eastern perimeter

### Pool House (80, -30)
- Recreation facilities
- Near main residence

## Paths

Major walkways connect:
1. Main Residence → Temple (primary path)
2. Main Residence → Guest Villas
3. Main Residence → Sundial
4. Main Residence → Helipad
5. Sundial → Temple (scenic route)

## Terrain Features

### Elevation
- Temple Hill: Highest point (~25m elevation)
- Main Residence area: Moderate elevation (~10m)
- Secondary hills in west and northeast
- Beach level: Sea level

### Beaches
- Southern shore: Sandy beaches
- Eastern shore: Rocky approach
- Northern cove: Dock area (not modeled in detail)

### Vegetation
- ~150 palm trees scattered throughout
- ~200 bushes and shrubs
- Denser vegetation in western area
